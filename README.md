# Onmyoji Notification

## Command

```
./onmyoji [COMMAND] [OPTIONS]

[COMMAND]
--daily Using to notify daily schedule.

[OPTIONS]
--time [time]: Using to notify with time in [COMMAND]

Examples:
./onmyji --daily --time 7
  Notify at 7 hour everyday with daily schedule.
```
